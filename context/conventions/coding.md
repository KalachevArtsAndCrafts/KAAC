# Coding Conventions

## Language

- Vanilla JavaScript (no TypeScript)
- ES modules (`import/export`)
- No framework — pure DOM manipulation

## Style

- Bundler: Parcel 1.x (CommonJS-compatible)
- CSS: plain CSS files, no preprocessor
- EditorConfig present (`.editorconfig`) — follow it

## Naming

- Files: `kebab-case.js`
- Folders: `kebab-case/`
- Functions/vars: `camelCase`

## Audio

- Sounds stored in `/sounds/`
- Format priority: `.mp3` → `.aac` → `.ogg`
- Muted variants suffixed with ` m` (e.g. `A1 m.mp3`)

## Git

- Main branch: `master`
- Versioning: semantic (`MAJOR.MINOR.PATCH`)
- Tags: `vMAJOR.MINOR.PATCH`
- Author: KalachevArtsAndCrafts
